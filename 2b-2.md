Q2: How can you view file content without a GUI editor?
Commands such as cat, less, or more can be used to view file contents directly in the terminal.

Q3: What is the difference between cp and mv?
cp creates a copy of a file while keeping the original.
mv moves or renames a file and removes it from the original location.
